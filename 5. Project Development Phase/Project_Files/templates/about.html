🎯 Project Objective

OptiCrop is designed to help farmers select the most suitable crop by analyzing Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH and Rainfall. The system applies Machine Learning to improve productivity, reduce farming risks, and support sustainable agriculture.
